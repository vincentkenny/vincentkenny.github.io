What to do?

The goal of the program is to search for the shortest path possible from the start position to the end goal.
You can move both the start position and the end goal by dragging or clicking and moving them.
You can add obstacle by clicking or dragging on the grid, or you can scatter random obstacles by clicking on the option before the Visualize Pathfinding button.
If you want to reset the board, there's also an option for that.
Whenever you're ready, start the algorithm by pressing the most obvious button. Have fun!
